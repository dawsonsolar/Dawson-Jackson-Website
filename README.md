# Dawson Jackson's Portfolio

Built with [Hugo](https://gohugo.io/) using the [LofiCode theme](https://github.com/meimakes/loficode-hugo-theme).

## Running locally

```bash
# 1. Clone the repo (with the theme submodule)
git clone --recurse-submodules https://github.com/dawsonsolar/Dawson-Jackson-Website.git
cd Dawson-Jackson-Website

# 2. Start the local server
hugo server
```

Then open http://localhost:1313 in your browser.
